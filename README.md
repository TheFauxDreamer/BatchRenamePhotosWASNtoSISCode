# BatchRenamePhotosWASNtoSISCode
Catchy name. A script that will rename photos from the WASN to the SIS code for bulk upload to Compass

## How to Use

1. Download a CSV from Compass > People Management, it MUST contain "Gov Code 2" & "Import Identifier" columns
2. Save the CSV & two script files alongside the photos you want to rename
3. Run the _Rename-StudentPhotos.bat_ file
4. Follow the prompts
5. You're all done
